# ACME Corp - Organization Standards

This directory contains organization-wide configuration and standards.

## What's Included

- **`.cursorrules`**: Company coding standards and best practices
- **`mcp.json`**: Approved MCP servers for AI assistance
- **`settings.yaml`**: Technical standards and requirements

## Usage

These configurations are automatically merged with team and personal settings.

Organization level has the **lowest priority** - team and personal settings will override these defaults.
